If you want to want reduce your RSS-Feed description value, you can cut it off after a given number of words.

== Description ==

In some cases it is usefull not to send too much content to your RSS-Feed. With this plugin you can cut it off after a given number.
At the optionpage (plugins -> Rss-Cutter) you can easily adjust how many words you want to have in the rss-feed-description.


== Installation ==

1. Upload `reduceRssFeed.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Adjust the number of words at Plugins -> Rss-Cutter optionpage

== Frequently Asked Questions ==

= What is the default value? =

4 Words
